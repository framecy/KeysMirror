# KeysMirror

**KeysMirror** 是一款 macOS 菜单栏工具，可将键盘按键或鼠标侧键映射为目标应用窗口内的鼠标点击。无需修改游戏或应用本身，一键触发任意位置的点击动作。

---

## 功能特性

- **按键 → 点击映射**：将键盘快捷键（支持修饰键组合）映射到应用窗口中的指定坐标
- **鼠标侧键支持**：右键、其他侧键均可作为触发器
- **无光标位移**：模拟点击时光标位置保持不变
- **覆盖层指示器**：半透明红点实时标记每个映射的点击位置，支持透明度调节
- **多应用配置**：为不同应用独立管理映射组，随前台应用自动切换
- **文字输入保护**：检测到文字输入焦点时自动暂停映射，不干扰正常打字
- **兼容 iOS-on-Mac 游戏**：通过 Apple Silicon 运行的 iPhone/iPad 游戏同样支持

---

## 系统要求

- macOS 13.0 Ventura 或更高版本
- Apple Silicon 或 Intel Mac
- 辅助功能权限（首次启动时引导授权）

---

## 安装

1. 从 [Releases](../../releases) 页面下载最新的 `KeysMirror.dmg`
2. 打开 DMG，将 `KeysMirror.app` 拖入 **应用程序** 文件夹
3. 首次启动时，按提示前往 **系统设置 → 隐私与安全性 → 辅助功能**，开启 KeysMirror 的权限
4. 菜单栏出现键盘图标，即表示运行正常

---

## 快速上手

### 1. 添加应用配置

点击菜单栏图标 → **打开配置** → **添加应用**，从正在运行的应用列表中选择目标应用。

### 2. 新建映射

选中应用配置后，点击 **新建映射**：

1. **标签名称**：给这条映射起个易识别的名字
2. **录制触发**：点击"录制触发"，按下想要绑定的键盘按键或鼠标侧键
3. **录制位置**：点击"录制位置"，程序会切换到目标应用，在目标窗口内**点击一次**想要触发点击的位置
4. 点击 **保存映射**

### 3. 使用映射

切换到目标应用，按下已绑定的快捷键，KeysMirror 会在录制的位置模拟一次左键点击。菜单栏图标会短暂变绿表示触发成功。

## 界面说明

| 状态栏图标 | 含义 |
|---|---|
| `⌨️👁` | 正常运行，映射已启用 |
| `⌨️` | 映射已禁用 |
| `⌨️…` | 辅助功能权限未授予 |
| 短暂绿色 | 映射触发成功 |

## 覆盖层指示器

在配置界面可为每个应用开启"显示快捷键指示器"，当目标应用处于前台时，会在每个映射的点击坐标处显示半透明红点，方便确认映射位置是否准确。

---

## 权限说明

KeysMirror 仅需 **辅助功能**（Accessibility）权限，用于：
- 监听全局键盘/鼠标事件（拦截触发键）
- 读取目标应用窗口位置（计算点击坐标）
- 向目标应用注入鼠标事件（模拟点击）

不会访问网络，不收集任何数据，所有配置保存在本地 `~/Library/Application Support/KeysMirror/mappings.json`。

## 常见问题

**Q：快捷键触发了但点击没效果？**  
A：打开配置 → 运行日志，确认"【执行动作】"日志是否出现。若出现则说明坐标可能偏移，建议重新录制位置。

**Q：在聊天框打字时快捷键误触发？**  
A：KeysMirror 会检测文字输入焦点并自动暂停映射，如遇不兼容的应用可临时通过菜单栏"禁用映射"手动关闭。

**Q：iOS 游戏不生效？**  
A：确认游戏已正常在 Apple Silicon Mac 上运行，并在配置中重新录制映射位置。

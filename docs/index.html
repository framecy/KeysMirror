核心功能

专为效率而生

每一个功能都经过深思熟虑，让映射操作无感融入你的工作流。

⌨️

按键映射点击

将任意键盘按键（支持 ⌘ ⇧ ⌥ ⌃ 等修饰键自由组合）绑定到应用窗口的指定坐标。按下即触发，响应延迟小于 5ms，比手动点击更快、更准。

🖱️

光标纹丝不动

模拟点击时鼠标指针完全静止，不影响当前正在进行的任何操作。

🎯

可视位置指示器

透明覆盖层在每个映射坐标处显示红色标记，实时确认映射位置是否准确。

📱

iOS 游戏兼容

完整支持通过 Apple Silicon 运行的 iPhone / iPad 游戏，iOS-on-Mac 架构无障碍。

🗂️

多应用独立配置

为不同应用单独管理映射组，切换前台应用时自动激活对应配置。

🛡️

隐私优先

完全离线，不联网，所有配置保存在本地。

🔇

文字输入智能保护

通过 Accessibility API 实时检测焦点元素类型。当光标落入聊天框、搜索栏等输入控件时，键盘映射自动静默，按键原样传递给应用；离开输入框后无需任何操作，映射即刻恢复——在游戏内打字与触发技能之间随意切换，零干扰。
